## 如何查看数据 🔍

### 方法1: 使用 Prisma Studio
```bash
cd backend
npm run prisma:studio
```
在浏览器打开 `http://localhost:5555`

### 方法2: 在前端查看
```bash
# 在项目根目录
npm run dev
```
1. 打开 `http://localhost:3000`
2. 在搜索框输入 `ヨルシカ` 或 `yorushika` 或 `夜鹿`
3. 点击搜索结果进入艺人页面
4. 查看10场公演信息

### 方法3: 使用 API
```bash
# 搜索艺人
curl http://localhost:3001/artists?search=ヨルシカ

# 获取公演列表
curl http://localhost:3001/lives?artistId=yorushika
```

## 常见问题 ❓

### Q: 如果数据已经存在怎么办？
A: 代码使用了 `upsert`，会自动更新现有数据而不是报错。

### Q: 如何重置数据库？
```bash
cd backend
npx prisma migrate reset
npm run seed:yorushika
```
⚠️ 警告：这会删除所有数据！

### Q: 如何修改数据？
编辑文件 `backend/prisma/seed-yorushika-2026.ts`，然后重新运行：
```bash
npm run seed:yorushika
```

### Q: 搜索不到夜鹿？
1. 确认数据已成功导入（检查终端输出）
2. 确认后端服务正在运行
3. 在浏览器控制台检查是否有错误

## 自定义数据 ✏️

如果你想修改票价、时间或其他信息：

1. 打开文件 `backend/prisma/seed-yorushika-2026.ts`
2. 找到对应的部分进行修改
3. 重新运行种子命令：
```bash
npm run seed:yorushika
```

### 修改票价示例
找到 `seatTypes` 部分：
```typescript
seatTypes: [
  { name: 'S席', price: '¥12,000', status: 'SOLD OUT' },
  { name: 'A席', price: '¥10,500' },
  { name: 'B席', price: '¥9,000' },
  { name: '注釈付き指定席', price: '¥8,500' },
],
```

### 添加搜索关键词示例
找到 `searchKeywords` 部分：
```typescript
searchKeywords: ['ヨルシカ', 'yorushika', '夜鹿', 'n-buna', 'suis'],
```
可以添加更多关键词，比如：`'夜に駆ける'`, `'だから僕は音楽を辞めた'` 等。
